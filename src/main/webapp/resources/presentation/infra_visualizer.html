보안 그룹명 허용 포트 (Inbound) 소스 (Source) 설명
ALB-SG 80, 443 0.0.0.0/0 전체 인터넷 접근 허용
EC2-SG 8080 (Tomcat) ALB-SG ALB를 통한 트래픽만 허용
RDS-SG 3306 (MySQL) EC2-SG 앱 서버(EC2) 접근만 허용
Redis-SG 6379 (Redis) EC2-SG 캐시 서버 접근만 허용

최소 권한 원칙 (Least Privilege)

각 계층은 바로 앞 단계의 트래픽만 허용하도록 구성하여 보안 위협을 최소화합니다.

네트워크 망분리

애플리케이션과 데이터베이스는 프라이빗 서브넷에 격리되어 있어, 외부 인터넷에서 직접 접근이 불가능합니다.
